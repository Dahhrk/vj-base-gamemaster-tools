# vj-base-gamemaster-tools

Enhance your Garry's Mod events with "vj-base-gamemaster-tools"! This project provides tools to automate event management, create dynamic battles, and improve roleplay experiences. Perfect for Gamemasters aiming to host engaging and immersive gameplay, the tools are designed for flexibility and adaptability to suit various event styles.

## Features

### Objectives System
Manage mission objectives dynamically with real-time synchronization to all players.

- **!objectives Command**: Type `!objectives` in chat to open the Objectives Manager
- **Full Management UI**: Add, edit, delete, and control objective visibility
- **Real-Time Sync**: Changes instantly appear for all players
- **Player Display**: Clean, minimal objectives panel for players
- **Permission-Based**: Admin-only access with OpenPermissions support

See [lua/players/README.md](lua/players/README.md) for detailed documentation.

### Dynamic NPC Spawner
Advanced NPC wave spawning system with AI behaviors, role-based NPCs, and vehicle support.

### UI Editor & Visualization
In-game UI editor for managing events with 3D world visualization and wave templates.

## Quick Start

### Objectives System
1. Join server as admin
2. Type `!objectives` in chat
3. Click "+ Add Objective" to create objectives
4. Players will see visible objectives automatically

For complete testing instructions, see [lua/players/TESTING_GUIDE.md](lua/players/TESTING_GUIDE.md).
